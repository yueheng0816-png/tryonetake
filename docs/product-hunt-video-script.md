## 录制操作要点

1. **窗口干净**：关掉其他标签页、通知、桌面杂乱图标
2. **鼠标流畅**：不要犹豫，操作路径提前演练两遍
3. **光标放大**：Loom 设置里有 "increase cursor size"，让观众看得清
4. **语速中等偏快**：1 分钟脚本约 170 词，别拖沓——宁可语速稍快也别超时
5. **重新录不要编辑**：这个长度完全不用剪辑，NG 了就从头录一遍
